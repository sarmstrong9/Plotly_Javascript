Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

Belly button bacteria taken from test subjects is displayed on a "per test subject" basis. This data is displayed/updated for each individual when "Test Subject ID No." is chosen from the dropdown menu.

The bar chart above displays the top 10 bacterial species (OTUs) and their corresponding number of samples for each subject ID. In addition, the "OTU Labels" are displayed when the mouse is hovered over each item.

The gauge chart above displays the recorded number of belly button washes for the chosen Subject ID.

The bubble chart above displays the complete dataset for each Subject ID. The OTU ID's are plotted on the x-axis, while the y-axis displays the number of samples. The bubble sizes also represent the number of samples to easily show the results for each OTU ID. In addition, the plotted datapoints and "OTU Labels" are displayed when the mouse is hovered over each item.
